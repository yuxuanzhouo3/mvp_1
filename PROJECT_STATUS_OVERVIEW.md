## 🔧 技术架构详情

### 前端技术栈
- **框架**: Next.js 14 (App Router)
- **语言**: TypeScript
- **样式**: Tailwind CSS
- **状态管理**: React Context + Hooks
- **UI 组件**: 60+ 自定义组件
- **路由**: Next.js 内置路由系统

### 后端技术栈
- **数据库**: Supabase (PostgreSQL)
- **缓存**: Redis (Upstash)
- **认证**: Supabase Auth + JWT
- **文件存储**: Supabase Storage
- **实时通信**: WebSocket + Supabase Realtime
- **API**: Next.js API Routes

### 第三方服务集成
- **AI 服务**: OpenAI GPT-4
- **支付处理**: Stripe + USDT + 支付宝
- **短信服务**: Twilio
- **部署平台**: Vercel
- **监控服务**: 内置监控 + 错误追踪
